What is GUSP?

The full name is Generic URL Shortening Protocol. It is a protocol that allows you to shorten URLs. It is based on my own idea, and it is trash

Protocol

Create a shortened URL

Request Header: Content-Type: application/gusp
Request Body: [gusp]URL|LENGTH-OF-URL|URL(|SHORTENED-ID)[/gusp]

Response Header: Content-Type: application/gusp
Response Body: [gusp]SUCCESS|LENGTH-OF-ID|SHORTENED-ID[/gusp]
OR
Response Body: [gusp]ERROR|LENGTH-OF-DESCRIPTION|ERROR-DESCRIPTION[/gusp]

Get a shortened URL

Request URL: https://gusp-based-service/SHORTENED-ID

Response Code: 302 Found
Response Header: Location: ORIGINAL-URL
